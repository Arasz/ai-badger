# 0.7.1 — Hermes external_dirs Registration + den-refresh Config Update

**Date:** 2026-07-24

## Fixed

- **scaffold.py**: `symlink_hermes_skills()` now registers the project's `.hermes/skills/` in `~/.hermes/config.yaml` `skills.external_dirs` so Hermes can discover ai-badger skills. Previously, symlinks were created but Hermes couldn't find them without manual config.
- **den-refresh**: `refresh.py` now updates `config.frameworkVersion` after a successful re-scaffold. Previously the config kept the original scaffold version.

## Added

- **Versioning rule**: `docs/changelog/` must be updated and VERSION bumped on every release.
- **Tests**: `test_scaffold_registers_hermes_external_dirs` and `test_scaffold_no_external_dirs_without_hermes` verify the external_dirs registration behavior.
