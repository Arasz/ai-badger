# 0.9.1 — GitHub Fallback for Framework Root Discovery

**Date:** 2026-07-24

## Fixed

- **`badger_lib.find_root()`** now falls back to `~/.ai-badger/framework/` (cached GitHub clone) when no local framework checkout is found. Previously, scripts would fail with "framework root not found" if run outside a framework repo.
- **`_ensure_framework_cache()`** — new helper that shallow-clones `https://github.com/Arasz/ai-badger` to `~/.ai-badger/framework/` or pulls latest if already cached.
- **All 6 scripts' `_bootstrap_lib()`** updated to try the cached framework when local detection fails.
- Users no longer need a local ai-badger checkout — the framework is auto-downloaded on first use.
