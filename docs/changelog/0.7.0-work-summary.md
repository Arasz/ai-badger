## What Changed

### Conceptual Changes
1. **Merged plugins into skills** — `plugins/` feature removed, replaced by `skills-source.json` + `skills.json` at each stack level
2. **Generalized skill sources** — `marketplaces.json` replaced by `skills-source.json` with `type` (marketplace/hub/tap/url/well-known) and `support` (common or per-agent array)
3. **Per-agent installation logic** — `plugins-instructions.json` declares how each agent installs external skills
4. **Hooks as first-class feature** — `features/common/hooks/` with `hooks-manifest.json` mapping hooks to agents
5. **Adjustments concept** — `features/{agent}/adjustments/` with per-feature scripts (`adjust_hooks.py`, `adjust_task.py`)
6. **Breaking version support** — `BREAKING_VERSIONS` file + den-refresh backs up `.ai-badger/` on breaking transitions
7. **pluginScope → skillScope** — config/manifest field renamed
