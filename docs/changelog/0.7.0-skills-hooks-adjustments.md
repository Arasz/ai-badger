# 0.7.0 — Skills Sources, Hooks, and Adjustments

**Status:** In progress  
**Date:** 2026-07-24  
**Spec:** `docs/specs/001-plugin-hooks-adjustments-refactor.md`

## Breaking Changes

- **`features/{stack}/plugins/` removed.** `marketplaces.json` and `plugins.json` replaced by `skills-source.json` and `skills.json` at the stack level. Re-scaffold affected projects.
- **`hooks/hooks.json` moved** to `features/common/hooks/hooks.json`.
- **`schemas/marketplaces.schema.json` and `schemas/plugins.schema.json` removed.** Replaced by `schemas/skills-source.schema.json` and `schemas/skills.schema.json`.
- **`config.json` `pluginScope` renamed** to `skillScope`.
- **`features/hermes/skills/task-extensions/hermes/extension.json` removed.** Replaced by `features/hermes/adjustments/adjustment.json`.

## Added

- **Skills sources** (`skills-source.json`): generalized external skill source declarations with `type` (marketplace, hub, tap, url, well-known) and `support` (common or per-agent array).
- **Hermes skill sources**: `hermes-hub` and `hermes-skills-tap` added to `features/hermes/skills-source.json`.
- **ai-badger as a skill source**: added to `features/common/skills-source.json`.
- **Per-agent plugin instructions** (`plugins-instructions.json`): declares how each agent installs external skills. Agents with no plugin system (copilot, junie) have empty instructions.
- **Hooks as first-class feature** (`features/common/hooks/`): unified location for Claude hooks.json, Hermes plugin hooks, and MCP index hooks. `hooks-manifest.json` maps hooks to agents.
- **MCP index auto-update hooks**: `mcp_index_hook.py` initializes index on session start and detects mcp commands for auto-rebuild.
- **Adjustments** (`features/{agent}/adjustments/`): schema-driven agent-level adaptations with per-feature scripts (`adjust_hooks.py`, `adjust_task.py`).
- **Changelog**: `docs/changelog/` directory with per-version entries.
- **New schemas**: `skills-source.schema.json`, `skills.schema.json`, `plugins-instructions.schema.json`, `adjustment.schema.json`, `hooks-manifest.schema.json`.
- **`scripts/install_plugins.py`**: orchestrates skill installation using per-agent instructions.

## Changed

- `badger_lib.py` FEATURES list: `plugins` removed, `hooks` and `adjustments` added.
- `index_build.py`: discovers hooks and adjustments features; skills-source/skills.json indexed under skills.
- `validate.py`: validates new schemas; runtime cross-reference checks for skills→skills-source.
- `index.schema.json`: `hooks` and `adjustments` added as valid feature keys, `plugins` removed.

## Migration

Clean break — no backward compatibility. Re-scaffold with `welcome-ai-badger` after upgrading.
